## Key Features

### 1. Novelty Validator
Don't build in the dark. We cross-reference your abstract against millions of GitHub repositories and research papers via the **Google Custom Search API**.
* **Uniqueness Score:** How original is your idea?
* **Pivot Suggestions:** AI suggests tweaks to make your project stand out.

### 2. Intelligent Roadmap Generator
Turn chaos into structure. Input your timeline (1 month, 3 months, etc.), and get a week-by-week engineering plan.
* **Curated Learning:** Automatically fetches relevant **YouTube Tutorials** for every specific tool in your stack using the YouTube Data API.

### 3. Stack Advisor
React or Vue? SQL or NoSQL?
* Get 3 tailored architecture options based on your team's skill level.
* Explains the "Why" behind every technical choice.

### 4. Viva Defender
The ultimate prep tool for students.
* Acts as an adversarial external examiner.
* Predicts tough technical questions based on your specific code logic.
* Provides "Ideal Expert Answers" to help you ace the Q&A.
